Grassroots racing deserves better than a Facebook post

Club and karting series still run on decades-old timing software, printed result sheets, and someone hoping to remember to update a spreadsheet.

🖥️

Results trapped on one laptop

Your timing export lives in one official's Orbits install. If they're not at the next event, nobody can find last month's results.

📄

PDFs and printed sheets

Drivers and families refresh a Facebook group hoping someone posts a photo of a printout taped to the timing trailer.

🧮

A standings spreadsheet, maybe

Championship points live in a spreadsheet someone updates when they remember — if the formula didn't break three rounds ago.

Upload a file. Get a site.

No IT skills required — designed for a volunteer with 20 minutes between heats.

Upload your timing export

Drop in an Orbits CSV export, or enter results by hand if there's no file. Nothing gets lost — the original file is stored before anything is even read.

Review what we read

See exactly what was parsed, fix anything that looks off, and add rows manually if needed. You publish, not a black box.

Publish and share

Get a clean, mobile-friendly results page with a shareable link — the kind you'd actually text a driver's parent.

Built in the open

Phase 0 — skeleton

One club, one uploaded file, one public results page

The core loop — sign in, create a season, upload a timing file, review, publish, and share a results link — is being built and demoed live against a fictional club, Blue Ridge Kart Club. See the README for exact build status.
